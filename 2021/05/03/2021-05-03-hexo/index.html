Hexo基本使用

发表于 2021-05-03 分类于 Hexo系列

关于Hexo静态博客的使用

Hexo博客创建

初始化Hexo项目

hexo init <项目名称>

安装Hexo项目依赖
  • cd <项目名称>
  • npm install
运行Hexo项目

hexo server

Hexo基本配置

title: 若华_Flower

安装Hexo Next主题

https://github.com/theme-next/hexo-theme-next/blob/master/docs/INSTALLATION.md

配置Next主题

theme: next

创建标签、分类、About

hexo new page about
